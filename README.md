# Vuelos App - Servicio Restful

Este repositorio contiene el servicio RESTful para la aplicación de gestión de vuelos. El servicio está desarrollado en PHP y es la parte del servicio de Vuelos App - Cliente Restful, sin esta no funcionará la otra.

## Tecnologías utilizadas

- **PHP:** Para el desarrollo del servidor y la lógica de la aplicación.
- **Base de datos:** Utiliza una base de datos (por ejemplo, MySQL) para almacenar la información de los vuelos y las reservas.
- **API RESTful:** El servicio proporciona una API RESTful para que los clientes interactúen con ella.

## Configuración

Para configurar y ejecutar el servicio, sigue estos pasos:

1. Clona este repositorio en tu servidor local o en un servidor web compatible con PHP.
2. Configura la conexión a la base de datos en el archivo de configuración correspondiente.
3. Asegúrate de que el servidor web tenga habilitada la ejecución de PHP.
4. Asegúrate de tener una base de datos adecuada y actualizada con la estructura necesaria para el servicio.
5. Para que funcione tal cuál con este código tienes que crear la carpeta restfulApiVuelos en la ruta C:\xampp\htdocs\_servWeb\
